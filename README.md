# Karaoke Finder

# Project Description

The Karaoke Finder App is a web application that allows users to search and play karaoke versions of their favourite songs. It uses the YouTube Data API to fetch real-time video results and display them in an interactive interface.

# API Used

* YouTube Data API v3

# Features

* Search karaoke songs by title or artist
* Display video results with thumbnails and embedded player
* Filter results based on keywords or recency
* Sort results alphabetically or by date
* Save favorite songs using local storage
* Responsive design for all devices

# API Usage

The application will use the YouTube Data API to fetch karaoke videos based on user input. When a user searches for a song, the application will send a request with the search query appended with the keyword "karaoke". The API response will include video details such as title, thumbnail, and video ID, which will be displayed dynamically on the webpage.

# Technologies Used

* HTML
* CSS
* JavaScript
* Fetch API

## How to Run the Project

1. Clone the repository
2. Add your YouTube API key in `script.js`
3. Open `index.html` in your browser
